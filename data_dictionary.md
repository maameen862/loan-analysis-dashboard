# 📘 Data Dictionary

| Column Name       | Description            |
| ----------------- | ---------------------- |
| Loan_ID           | Unique loan identifier |
| Customer_ID       | Unique customer ID     |
| Loan_Amount       | Applied loan amount    |
| Sanctioned_Amount | Approved loan amount   |
| Disbursed_Amount  | Disbursed amount       |
| Recovery_Amount   | Amount recovered       |
| Loan_Product      | Type of loan           |
| Channel           | Source of application  |
| Branch            | Branch location        |
| Date              | Application date       |
